# Cube-desk-app

<br>

## 소개
맥의 알림센터 기능을 보고 영감을 받았으며, 평소에 주로쓰는 기능들을 한 페이지에 담아 탭에 띄어놓고 빨리 사용할 수 있게 만들어 보았습니다.

<br>

## 구현 기능
  - Profile
  - Date & Time
  - Temperture & Weather
  - Calculator
  - Drawing-pad 
  - To do list
